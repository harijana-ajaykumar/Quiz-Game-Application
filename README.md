# Quiz Game Application

## 📌 Project Overview
The Quiz Game Application is a simple web-based quiz project developed using HTML, CSS, and JavaScript. It allows users to answer multiple-choice questions, calculates the score, and displays the final result at the end of the quiz.

## 🚀 Features
- Multiple Choice Questions
- Interactive User Interface
- Score Calculation
- Final Result Display
- Easy Navigation Between Questions

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
```
Quiz-Game/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 How It Works
1. The quiz starts with the first question.
2. The user selects an answer.
3. The application checks whether the answer is correct.
4. The score is updated automatically.
5. After all questions are completed, the final score is displayed.

## 📸 Output
- Displays quiz questions and options.
- Calculates score based on correct answers.
- Shows final result after quiz completion.
